# Spatial AI Console

A premium, high-fidelity spatial AI dashboard built with React Native, Expo, and Three.js (@react-three/fiber).

## 🚀 Key Features
- **Spatial Grid Engine**: Interact with up to 9 AI models simultaneously in a physics-based 3D grid.
- **Hybrid 3D/2D UI**: PBR refractive glass materials for AI cards with high-performance opaque control overlays.
- **Multi-Modal Suites**: Specialized workflows for General Chat, Image Generation, Video Synthesis, Audio/Music, and Coding.
- **Consensus Engine**: "Collide" active models to synthesize agreement and map dissenting logic spatially.
- **Store-Ready MVP**: Includes bundle IDs, privacy permissions, mock IAP flows, and realistic auth simulations.

## 🛠 Setup & Installation
1. Clone the repo.
2. Run `npm install`.
3. Create a `.env` file in the root:
   ```
   EXPO_PUBLIC_GROQ_API_KEY=your_key_here
   EXPO_PUBLIC_OPENROUTER_API_KEY=your_key_here
   ```
4. Run `npx expo start`.
